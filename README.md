## 项目介绍
ShardingSphere分库分表、读写分离、数据脱敏、多租户实践案例

>欢迎star和fork,支持一下作者，让更多人发现

## 软件架构
|  技术栈   |  版本 |
| --- | --- |
|  jdk |   1.8.0_161  |
|  SpringBoot   | 2.2.2.RELEASE   |
|  Shardingsphere   | 4.0.1   |
|  Mybatis-plus   | 3.3.0   |
|  Hutool   | 5.0.6   |
|  HikariCP   | 2.7.9  |

## 安装教程
* 先建mysql实例
* 初始化sql,sql会上传到群文件里面
* 除了common包，均可直接启动（启动时需要配置dbIp变量）
